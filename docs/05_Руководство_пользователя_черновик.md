## 1. ВВЕДЕНИЕ

DotElectric Template Editor — инструмент для создания шаблонов конструкторских документов (рамки, штампы, текстовые поля) в соответствии с ГОСТ ЕСКД.

### 1.1. Возможности

- Создание шаблонов форматов A0-A4 и пользовательских
- Рисование линий, прямоугольников, текста
- Привязка к координатной сетке
- Сохранение в формат .tdel
- Печать шаблонов

### 1.2. Системные требования

- ОС: Windows 10/11
- .NET 10 Runtime
- Разрешение экрана: от 1280×720

## 3. СОЗДАНИЕ ШАБЛОНА

### 3.1. Новый шаблон

1. Выберите **Файл → Создать** или нажмите `Ctrl+N`
2. В диалоге выберите формат листа:
   - **Стандартные:** A0, A1, A2, A3, A4
   - **Полуторные (×2):** A0×2 (2378×841 мм), A1×2 (1188×841 мм), A2×2 (840×594 мм), A3×2 (594×420 мм), A4×2 (420×297 мм)
   - **Пользовательский:** до 2000×2000 мм
3. Нажмите **OK**

### 3.2. Открытие существующего

1. Выберите **Файл → Открыть** или нажмите `Ctrl+O`
2. Выберите файл `.tdel`
3. Нажмите **Открыть**

## 4. ИНСТРУМЕНТЫ РИСОВАНИЯ

### 4.1. Линия

**Назначение:** Рисование прямых линий (границы, рамки)

**Использование:**
1. Выберите инструмент **Линия** на панели инструментов
2. Нажмите левую кнопку мыши в начальной точке
3. Перетащите в конечную точку
4. Отпустите кнопку мыши

**Совет:** При включённой привязке концы линии автоматически выравниваются по сетке.

---

### 4.2. Прямоугольник

**Назначение:** Создание рамок, областей штампа

**Использование:**
1. Выберите инструмент **Прямоугольник**
2. Нажмите левую кнопку мыши в левом верхнем углу
3. Перетащите в правый нижний угол
4. Отпустите кнопку мыши

---

### 4.3. Текст

**Назначение:** Добавление текстовых надписей

**Использование:**
1. Выберите инструмент **Текст** (клавиша `T`)
2. Кликните в месте размещения текста
3. Для редактирования содержимого сделайте **двойной клик** по тексту
4. В открывшемся редакторе:
   - **Ctrl+Enter** — завершить редактирование и закрыть редактор
   - **Enter** — вставка новой строки (если включён MultiLine)
   - **Escape** — отменить изменения и закрыть редактор

**Поворот:**
- В панели свойств укажите угол поворота в градусах (0-359)
- Поворот происходит по часовой стрелке вокруг опорной точки

**Шрифты:**
- **ГОСТ А** (узкий) — по умолчанию
- **ГОСТ Б** (широкий)
- Выбор в панели свойств (Параметры → Шрифт)

**Shift + drag** — при рисовании линии ограничивает угол до 45°, при рисовании прямоугольника — сохраняет пропорции (квадрат).

## 5. РАБОТА С ОБЪЕКТАМИ

### 5.1. Выделение

- **Одиночное выделение:** Клик по объекту
- **Множественное выделение:** `Shift` + клик по объектам
- **Выделение рамкой:** Перетаскивание с зажатой ЛКМ

### 5.2. Перемещение

1. Выделите объект
2. Зажмите ЛКМ и перетащите
3. Отпустите кнопку

### 5.3. Удаление

1. Выделите объект
2. Нажмите `Delete`

### 5.4. Копирование

1. Выделите объект
2. Нажмите `Ctrl+C`
3. Нажмите `Ctrl+V`
4. Переместите копию на нужное место

### 5.5. Отмена и повтор

- **Отмена:** `Ctrl+Z`
- **Повтор:** `Ctrl+Y`

## 6. СЕТКА И ПРИВЯЗКА

### 6.1. Включение/отключение сетки

- **Кнопка:** Нажмите кнопку **Сетка** на панели инструментов
- **Горячая клавиша:** `F7`

### 6.2. Выбор шага сетки

1. Откройте **Вид → Сетка → Шаг**
2. Выберите шаг:
   - **0.5 мм** — высокая точность
   - **1 мм** — стандартный
   - **5 мм** — крупная
   - **10 мм** — очень крупная

### 6.3. Привязка к сетке

- **Включение:** Кнопка **Привязка** или `F8`
- При включённой привязке объекты «прилипают» к узлам сетки

## 8. СОХРАНЕНИЕ И ЗАГРУЗКА

### 8.1. Сохранение

1. Выберите **Файл → Сохранить** или нажмите `Ctrl+S`
2. Введите имя файла
3. Выберите папку
4. Нажмите **Сохранить**

Файл сохранится с расширением `.tdel`

### 8.2. Сохранение как

1. Выберите **Файл → Сохранить как**
2. Введите новое имя
3. Нажмите **Сохранить**

### 8.3. Формат .tdel

Файл `.tdel` — это ZIP-архив, содержащий `template.xml` с описанием шаблона.

**Внутренняя структура:**
```
template.tdel
└── template.xml
```

**Особенности формата:**
- Все координаты и размеры хранятся в **микронах** (целые числа). 1 мм = 1000 микрон.
- Это обеспечивает **полную точность** — при сохранении и загрузке данные не теряются.
- XML можно открыть любым текстовым редактором для просмотра.
- Формат соответствует схеме `http://dotelectric.local/schema/template`.

## 9. ПЕЧАТЬ

### 9.1. Печать шаблона

1. Выберите **Файл → Печать** или нажмите `Ctrl+P`
2. Выберите принтер
3. Настройте параметры (количество копий, диапазон)
4. Нажмите **Печать**

### 9.2. Предварительный просмотр

Приложение имеет встроенный предпросмотр печати на основе WPF DocumentViewer.

1. Выберите **Файл → Предпросмотр печати** или нажмите `Ctrl+Shift+P`
2. В окне предпросмотра отображается лист с нарисованными объектами (линии, прямоугольники, текст)
3. Сетка, маркеры выделения и preview-фигуры НЕ отображаются в предпросмотре
4. Используйте встроенную панель DocumentViewer для:
   - **Масштабирования** — ползунок зума или кнопка FitToWidth
   - **Печати** — кнопка Print открывает стандартный диалог Windows
   - **Навигации** — прокрутка и страницы (для больших листов)
5. Закройте окно предпросмотра (X / Close), чтобы вернуться к редактированию

## 10. НАСТРОЙКИ

### 10.1. Диалог настроек

**Открытие:** **Файл → Настройки...**

Диалог настроек содержит следующие разделы:

**ОФОРМЛЕНИЕ**
- **Тема:** выбор светлой (Light) или тёмной (Dark) темы оформления

**СЕТКА**
- **Показывать:** отображение сетки на холсте по умолчанию
- **Привязка:** привязка объектов к узлам сетки
- **Шаг (мм):** шаг сетки в миллиметрах

**АВТОСОХРАНЕНИЕ**
- **Интервал:** периодичность автосохранения в минутах (1–60)

**НОВЫЙ ШАБЛОН**
- **Формат:** формат листа по умолчанию (A0–A4, A4×2–A0×2)
- **Масштаб:** масштаб отображения по умолчанию (25%–200%)

Изменения вступают в силу после нажатия кнопки «Сохранить».

### 10.2. Тема оформления

- **Быстрое переключение:** **Вид → Тема → Светлая/Тёмная**
- **Горячая клавиша:** `F9`

### 10.3. Масштаб

- **Увеличить:** `Ctrl++` или кнопка 🔍+
- **Уменьшить:** `Ctrl+-` или кнопка 🔍-
- **Вместить:** `Ctrl+0` или кнопка Fit
- **Выбор из списка:** выпадающий список масштаба в панели инструментов (10%–1000%)

### 10.4. Библиотека шаблонов

Шаблоны хранятся в папке:
```
%APPDATA%\DotElectric\Templates\
```

При запуске приложения папка сканируется автоматически. Шаблоны отображаются в левой панели в алфавитном порядке (имя файла без `.tdel`). Если шаблонов нет — панель пустая.

**Добавление шаблона:** нажмите кнопку **«Импорт»** в левой панели и выберите файл `.tdel`.
**Удаление шаблона:** выберите шаблон в списке и нажмите кнопку **«Удалить»**.
