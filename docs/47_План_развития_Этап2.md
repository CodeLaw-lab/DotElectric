## Sprint 27 — Архитектурный рефакторинг (неделя 1) ✅ ЗАВЕРШЁН

### Цель ✅
Уменьшить сложность кодовой базы, подготовить к Этапу 2.

### Задачи ✅

#### 1. Декомпозиция EditorViewModel (13 SP) ✅

**Проблема:** EditorViewModel — 1037 строк, отвечает за zoom, pan, selection, clipboard, undo, grid, tools, inline editing, status bar.

**Решение:** Выделено 4 менеджера:

| Менеджер | Ответственность | Строк |
|----------|-----------------|-------|
| `ZoomPanManager` | Zoom, PanOffset, FitToScreen, ZoomIn/Out | ~110 |
| `SelectionManager` | SelectedObjects, SelectionBox, hover, Add/Remove/Single | ~60 |
| `ClipboardManager` | Copy, Paste, Cut, _clipboard list | ~30 |
| `ToolManager` | ActiveTool, PushTool/PopTool, GetOrCreateTool, Reset | ~70 |

**EditorViewModel после:** ~784 строк (координация + Template + CommandHistory)

**Тесты:** +23 новых теста для менеджеров

**Результат:** ✅ Все 1394 теста проходят, 0 сбоев

#### 2. Разделение CommonConverters (3 SP) ⏸ НЕ ВЫПОЛНЕНО
