### 2.1. ISettingsService + SettingsService

**Файлы:** `Services/ISettingsService.cs`, `Services/SettingsService.cs`, `Services/AppSettings.cs`

| Метод | Описание |
|-------|----------|
| `Load()` | Загрузить настройки из `%APPDATA%\DotElectric\settings.json` |
| `Save(AppSettings)` | Сохранить настройки в JSON |
| `Get<T>(key, defaultValue)` | Получить настройку по ключу |
| `Set<T>(key, value)` | Установить настройку (авто-сохранение) |

**AppSettings — известные настройки:**

| Свойство | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `AutosaveIntervalMinutes` | int | 5 | Интервал автосохранения |
| `Theme` | string | "Light" | Тема оформления |
| `ShowGrid` | bool | true | Показывать сетку |
| `SnapToGrid` | bool | true | Привязка к сетке |
| `GridStepMm` | double | 5.0 | Шаг сетки (мм) |
| `DefaultZoom` | double | 1.0 | Масштаб по умолчанию |
| `DefaultSheetFormat` | string | "A3" | Формат листа |
| `CustomSettings` | Dictionary | new() | Дополнительные настройки |

**Особенности:**
- Кэширование настроек (`_cachedSettings`) — файл читается один раз
- Graceful degradation: при повреждённом JSON — настройки по умолчанию
- Поддержка произвольных ключей через `CustomSettings`

### 2.3. ITemplateService + TemplateService

**Файлы:** `Services/ITemplateService.cs`, `Services/TemplateService.cs`

| Метод | Описание |
|-------|----------|
| `CreateNew(format)` | Создать пустой шаблон (A0-A4, Custom) |
| `Load(filePath)` | Загрузить из .tdel (ZIP → XML → Template) |
| `Save(template, filePath)` | Сохранить в .tdel (Template → XML → ZIP) |
| `Validate(template)` | Валидация через ValidationService → `IEnumerable<string>` |

**Формат .tdel:**
- ZIP-архив с одним файлом: `template.xml`
- XML-сериализация через `XmlSerializer` (DTO-паттерн)
- UTF-8 кодировка, indent

**DTO-классы:**
- `TemplateDto` — корень XML (Version, Metadata, Sheet, Objects)
- `MetadataDto` — метаданные
- `SheetDto` — параметры листа
- `ObjectDto` — универсальный объект (Line/Rectangle/Text)

**Mapping:**
- `MapToTemplate(dto)` — DTO → Model
- `MapToDto(template)` — Model → DTO
- `MapToObject(dto)` — DTO → ITemplateObject (по типу)
- `MapToDto(obj)` — ITemplateObject → DTO (pattern matching)

**Особенности:**
- `Save` обновляет `Metadata.ModifiedDate`
- Создаёт директорию, если не существует
- Удаляет старый файл перед записью
- Валидация через `ValidationService` (только Error-уровень)

### 2.5. AutosaveService

**Файлы:** `Services/AutosaveService.cs`

| Метод/Свойство | Описание |
|----------------|----------|
| `Start()` | Запустить таймер (интервал из настроек, по умолч. 5 мин) |
| `Stop()` | Остановить таймер |
| `AutosaveAllTabsAsync(tabs, ct)` | Сохранить все «грязные» вкладки |
| `LoadSession()` | Загрузить `session.json` (для восстановления) |
| `GetAutosaveFilePath(fileName)` | Полный путь к автосохранённому файлу |
| `ClearAutosaveFolder()` | Удалить все файлы автосохранения |
| `AutosaveTick` | Событие тика таймера (подписка MainViewModel) |
| `IDisposable` | Корректное освобождение ресурсов |

**Модели:**
- `AutosaveSession` — SessionStart, LastAutosave, Tabs
- `AutosaveTabInfo` — TabId, DisplayName, OriginalFilePath, AutosaveFile, WasDirty
- `IAutosaveTab` — минимальный интерфейс для EditorViewModel (TabId, FilePath, DisplayName, IsDirty, Template)

**Особенности:**
- **SemaphoreSlim(1, 1)** — защита от race condition
- **Папка:** `%APPDATA%\DotElectric\autosave\`
- **Имя файла:** `autosave_{tabId}_{yyyyMMdd_HHmmss}.tdel`
- **session.json** — метаданные сессии (удаляется, если нет «грязных» вкладок)
- **Очистка:** удаляет старые файлы вкладки (оставляет последний)
- **Очистка:** удаляет файлы старше 7 дней
- **Логирование:** Debug (успех), Error (сбой), Warning (очистка)

**Поток автосохранения:**
```
Timer tick → AutosaveTick event → MainViewModel вызывает AutosaveAllTabsAsync
→ SemaphoreSlim.WaitAsync → для каждой «грязной» вкладки:
  → ITemplateService.Save(template, filePath)
  → CleanupOldAutosaveForTab(tabId)
  → Добавить в AutosaveSession
→ Сохранить session.json
→ CleanupOldAutosaveFiles()
→ SemaphoreSlim.Release
```
