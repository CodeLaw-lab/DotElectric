| Fixed-Point координаты (микроны) | Исключение погрешности float при round-trip |

| Serilog (Debug, rolling file, 30 файлов) | Полная трассировка без потери контекста |
| Автосохранение каждые 5 мин с session.json | Восстановление после сбоев |
| Multi-tab через ObservableCollection<EditorViewModel> | Параллельная работа с несколькими шаблонами |
| Один проект MVP → разделение к Этапу 3-4 | Быстрый старт, чистая архитектура с начала |
| Сетка: Template.DefaultGrid = const 5мм, EditorViewModel.GridSettings = настраивается | Разделение константы и настроек пользователя |

### Критически важные правила

| Правило | Контроль |
|---------|----------|
| Models НЕ импортируют System.Windows.* | Code review, статический анализ |
| I*Service НЕ содержат UI-типы | Code review |
| ICommand (Undo) ≠ System.Windows.Input.ICommand | Разные namespace |
| Чистые зависимости: Models ← Services ← VM ← Views | Запрет циклических ссылок |
| Preview reuse: НЕ создавать новый Shape при MouseMove | Performance критично |
