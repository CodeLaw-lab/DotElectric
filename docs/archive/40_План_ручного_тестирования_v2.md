## 1. Введение

### 1.1. Цель документа

Настоящий план определяет стратегию ручного тестирования CAD-системы DotElectric (Этап 1 — Редактор шаблонов листов) после завершения Sprint 22.

### 1.2. Область применения

Документ охватывает **всю реализованную функциональность** включая:
- Управление шаблонами (multi-tab, ориентация листа)
- Рисование объектов (линии, прямоугольники, текст)
- Выделение и манипуляции (drag, resize, rotate, delete)
- Undo/Redo (50 уровней)
- Сетка и привязка (GridNodesLayer, шаг сетки)
- Панель свойств (16 RelayCommands + 15 *FromString обёрток)
- Сохранение и загрузка (.tdel, XML в ZIP)
- Масштабирование и навигация (Zoom, Pan, Fit to Screen)
- Печать
- Темы оформления (Light/Dark)
- Горячие клавиши (24 штуки)
- Inline editing текста
- WeakReferenceMessenger (закрытие вкладок)
- IApplicationLifecycle

### 1.3. Критерии приёмки

| Уровень | Критерий |
|---------|----------|
| ✅ **PASS** | Все P0 и P1 тесты пройдены, ≤ 3 P2 провалены |
| ⚠️ **MARGINAL** | Все P0 пройдены, ≤ 2 P1 провалены |
| ❌ **FAIL** | Любой P0 провален, или > 2 P1 провалены |

### 1.4. Известные проблемы (НЕ тестируются)

Следующие проблемы **известны и запланированы к исправлению в Sprint 23**. При тестировании **НЕ считать их багами нового кода**:

#### P0 — Критические (6 задач, 15 SP)

| # | Проблема | Влияние на тестирование |
|---|----------|-------------------------|
| 1 | **MouseBehavior static state** — глобальные `_clickCount`/`_lastClickTime` общие для всех элементов | Двойной клик на одном элементе может сработать на другом. При тестировании double-click **допускать** ложные срабатывания |
| 2 | **PreviewLineChangedBehavior утечка** — подписка на PropertyChanged без отписки | Утечка памяти при открытии/закрытии вкладок. **Не проверять** утечки памяти в рамках этого плана |
| 3 | **EditorCanvasBehavior._states утечка** — нет cleanup при Unloaded | Утечка Canvas + State при закрытии вкладок. **Не проверять** утечки памяти |
| 4 | **SettingsService молча глотает исключения** | Потеря настроек без уведомления. При тестировании настроек **фиксировать** случаи потери |
| 5 | **AutosaveService.AutosaveTick из ThreadPool** | Cross-thread risk. При тестировании автосохранения **фиксировать** исключения |
| 6 | **PreviewLineChangedBehavior._canvasRefs без синхронизации** | Race condition при одновременном открытии/закрытии вкладок. **Фиксировать** редкие сбои |

#### P1 — Серьёзные (6 задач, 16 SP)

| # | Проблема | Влияние на тестирование |
|---|----------|-------------------------|
| 7 | **ITool зависит от WPF** (MouseButton, ModifierKeys) | Невозможно unit-тестировать без WPF. **Не влияет** на ручное тестирование |
| 8 | **ThemeService использует Serilog.Log напрямую** | Невозможно мокать в тестах. **Не влияет** на ручное тестирование |
| 9 | **IPrintService.Print(Template) всегда бросает** | Нарушение LSP. При тестировании печати **ИСПОЛЬЗОВАТЬ** только `PrintWithVisual` |
| 10 | **SettingsService нет thread safety** | Race condition при одновременном Load/Save. **Фиксировать** редкие потери настроек |
| 11 | **TabItemMiddleClickBehavior — reflection** | Хрупкий, нет compile-time проверки. При тестировании middle-click **фиксировать** сбои |
| 12 | **TextTool hardcoded значения** | fontSize=3500, font="ГОСТ Б", content="Текст" не настраиваются. **Ожидаемое поведение** |

#### P2 — Улучшения (4 задачи, 8 SP)

| # | Проблема | Влияние |
|---|----------|---------|
| 13 | Дублирование `SnapIfEnabled` в 3 инструментах | DRY нарушение. **Не влияет** на функциональность |
| 14 | Дублирование bounding box текста | DRY нарушение. **Не влияет** на функциональность |
| 15 | `ValidationService.ValidateHexColors()` — мёртвый код | Ложное ощущение функциональности. **Не проверять** валидацию цветов |
| 16 | Нет ILogger в TemplateService, FileService, PrintService | Невозможно диагностировать проблемы. **Фиксировать** ошибки без логов |

## 8. Глоссарий

| Термин | Определение |
|--------|-------------|
| **микроны** | 1 мм = 1000 микрон, единица измерения в модели |
| **Fixed-Point** | Координаты хранятся как `long`, НЕ `double` |
| **Round-trip** | Сериализация/десериализация без потерь |
| **Декартова система** | (0,0) = левый нижний угол, Y↑ |
| **WPF-координаты** | (0,0) = левый верхний угол, Y↓ |
| **Preview** | Предварительная отрисовка при рисовании |
| **Hit-testing** | Определение объекта под курсором |
| **Nudge** | Сдвиг стрелками |
| **PushTool/PopTool** | Временная смена инструмента (ToolStack) |
| **LTR/RTL** | LeftToRight / RightToLeft (selection box) |
| **IsDirty** | Флаг несохранённых изменений |
| **CommandHistory** | 50 уровней undo/redo |
| **BatchCommand** | Составная операция (undo в LIFO) |
| **WeakReferenceMessenger** | Коммуникация ViewModel без циклических зависимостей |
| **GridNodesLayer** | Слой узлов сетки (точки, RenderTransform) |
| **Inline editing** | Двойной клик на тексте → TextBox overlay |
