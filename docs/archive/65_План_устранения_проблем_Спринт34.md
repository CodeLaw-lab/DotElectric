### P1-C1: Preview-паттерн в DrawingLineTool

**Файл:** `src/DotElectric.TemplateEditor/Tools/DrawingLineTool.cs:67-70`

2. В `OnMouseDown` при первом клике создать `_previewLine = new Line(...)` и присвоить `_editor.PreviewLine = _previewLine`
3. В `OnMouseMove` обновлять ТОЛЬКО свойства `_previewLine.StartMicronsX/Y`, `_previewLine.EndMicronsX/Y` без создания нового объекта
4. В `Reset()` и при финализации — обнулять `_previewLine = null`

### P2-M1: DeleteObjectCommand с нестабильным индексом

**Файл:** `src/DotElectric.TemplateEditor/Commands/DeleteObjectCommand.cs:32`
**Проблема:** Индекс захватывается в конструкторе, не отражает состояние коллекции на момент Undo.

**Решение:** Хранить ссылку на объект, в Undo вставлять на позицию после проверки.

**Изменение:**
```csharp
// Удалить _index, хранить только _object
// В Undo:
_objects.Insert(Math.Min(_objects.Count, _originalIndex), _object);
```
Где `_originalIndex` захватывается в момент Execute, а не в конструкторе.

Лучший вариант — захватывать индекс в Execute:
```csharp
private int _executedIndex = -1;

public void Execute()
{
    _executedIndex = _objects.IndexOf(_object);
    _objects.Remove(_object);
    _markDirty?.Invoke();
}

public void Undo()
{
    var insertAt = Math.Clamp(_executedIndex, 0, _objects.Count);
    _objects.Insert(insertAt, _object);
}
```

### P3-D3: ToolManager.GetOrCreateTool через словарь

**Файл:** `src/DotElectric.TemplateEditor/ViewModels/Managers/ToolManager.cs:30-45`
**Решение:** Заменить цепочку `if` на `Dictionary<Type, Func<ITool>>`.

**Изменение:**
```csharp
private readonly Dictionary<Type, Func<ITool>> _toolFactories;
private readonly Dictionary<Type, ITool> _toolCache = new();

public T GetOrCreateTool<T>() where T : class, ITool
{
    if (_toolCache.TryGetValue(typeof(T), out var cached))
        return (T)cached;

    if (!_toolFactories.TryGetValue(typeof(T), out var factory))
        throw new ArgumentException($"Unknown tool type: {typeof(T)}");

    var tool = factory();
    _toolCache[typeof(T)] = tool;
    return (T)tool;
}
```
