## 1. Подготовка к тестированию

### 1.1 Требования
- [ ] Windows 10/11
- [ ] .NET 10 SDK установлен
- [ ] Исходный код клонирован в `D:\dotElectricTest`

### 1.2 Сборка приложения
```bash
cd D:\dotElectricTest
dotnet build src/DotElectric.TemplateEditor.slnx
dotnet run --project src/DotElectric.TemplateEditor
```

### 1.3 Ожидаемый результат
- ✅ Приложение запускается без ошибок
- ✅ Главное окно отображается корректно
- ✅ Нет предупреждений в консоли (кроме ожидаемых)
