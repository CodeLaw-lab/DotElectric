## 1. Подготовка к тестированию

### 1.1 Требования к окружению
- [ ] Windows 10/11 64-bit
- [ ] .NET 10 SDK установлен
- [ ] Исходный код клонирован в D:\dotElectricTest
- [ ] Разрешение экрана не менее 1280x1024 (рекомендуется 1920x1080)
- [ ] Шрифты ГОСТ А/Б установлены (встроены в Resources/Fonts)

### 1.2 Сборка и запуск
```
cd D:\dotElectricTest
dotnet build src/DotElectric.TemplateEditor.slnx
dotnet run --project src/DotElectric.TemplateEditor
```

### 1.3 Ожидаемый результат сборки
- [ ] 0 ошибок, 0 предупреждений
- [ ] 1760+ тестов пройдены, 1 pre-existing skip

### 1.4 Обозначения
- **P0** - Критический (blocker для релиза)
- **P1** - Высокий (важная функциональность)
- **P2** - Средний (желательно протестировать)
- Результат: PASS / FAIL / SKIP / KNOWN
