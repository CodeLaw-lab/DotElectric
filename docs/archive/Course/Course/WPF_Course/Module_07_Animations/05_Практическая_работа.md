# Модуль 7: Практическая работа

## Задание: Создание анимированного UI

**Время выполнения:** 3 часа  
**Уровень:** Продвинутый  
**Максимальный балл:** 100

---

## Описание

Необходимо создать анимированный пользовательский интерфейс с использованием Storyboard, Timeline-анимаций и Easing Functions.

---

## Требования

### 1. Анимация появления (Fade In) (30 минут)

Создайте анимацию появления окна:
- Все элементы появляются с Opacity=0 → Opacity=1
- Sequential animation (каждый следующий через 0.1 сек)
- QuadraticEase EaseOut

```xml
<Storyboard x:Key="FadeInStoryboard">
    <!-- Элементы появляются последовательно -->
</Storyboard>
```

---

### 2. Анимация перемещения (45 минут)

Создайте анимацию перемещения объекта:
- Прямоугольник перемещается по клику
- От текущей позиции к точке клика
- Canvas.Left и Canvas.Top одновременно
- Duration 0.5 секунды
- QuadraticEase EaseOut

```csharp
private void Canvas_MouseDown(object sender, MouseButtonEventArgs e)
{
    var point = e.GetPosition(myCanvas);
    AnimateRectToPoint(point.X, point.Y);
}
```

---

### 3. Анимация цвета (30 минут)

Создайте анимацию изменения цвета:
- Button меняет цвет при наведении
- ColorAnimation Background
- EnterActions (hover) и ExitActions (unhover)
- Duration 0.2 секунды

```xml
<Style x:Key="ColorChangeButton" TargetType="Button">
    <Setter Property="Template">
        <Setter.Value>
            <ControlTemplate TargetType="Button">
                <Border x:Name="border" Background="{TemplateBinding Background}">
                    <ContentPresenter/>
                </Border>
                
                <ControlTemplate.Triggers>
                    <Trigger Property="IsMouseOver" Value="True">
                        <Trigger.EnterActions>
                            <!-- ColorAnimation -->
                        </Trigger.EnterActions>
                        <Trigger.ExitActions>
                            <!-- ColorAnimation back -->
                        </Trigger.ExitActions>
                    </Trigger>
                </ControlTemplate.Triggers>
            </ControlTemplate>
        </Setter.Value>
    </Setter>
</Style>
```

### 4. Pulsing Animation (30 минут)

Создайте пульсирующую анимацию:
- Круг пульсирует (ScaleTransform)

- AutoReverse=True

- SineEase или QuadraticEase

```xml
<Ellipse Width="100" Height="100" Fill="Red">
    <Ellipse.RenderTransform>

    </Ellipse.RenderTransform>
    
    <Ellipse.Triggers>
        <!-- Storyboard с пульсацией -->
    </Ellipse.Triggers>
</Ellipse>
```

### 6. Bouncing Ball (45 минут)

Создайте анимацию прыгающего мяча:
- Ellipse падает сверху
- BounceEase при ударе о "пол"
- Горизонтальное перемещение
- RepeatBehavior=Forever

```xml
<Canvas Width="400" Height="300">
    <Ellipse x:Name="bouncingBall"
             Width="40" Height="40"
             Fill="Orange"
             Canvas.Left="0" Canvas.Top="0">
        <Ellipse.Triggers>
            <EventTrigger RoutedEvent="Loaded">
                <BeginStoryboard>
                    <Storyboard RepeatBehavior="Forever">
                        <!-- Vertical bounce -->
                        <DoubleAnimation Storyboard.TargetName="bouncingBall"
                                         Storyboard.TargetProperty="(Canvas.Top)"
                                         From="0" To="260"
                                         Duration="0:0:1">
                            <DoubleAnimation.EasingFunction>
                                <BounceEase Bounces="3" EasingMode="EaseOut"/>
                            </DoubleAnimation.EasingFunction>
                        </DoubleAnimation>
                        
                        <!-- Horizontal move -->
                        <DoubleAnimation Storyboard.TargetName="bouncingBall"
                                         Storyboard.TargetProperty="(Canvas.Left)"
                                         From="0" To="360"
                                         Duration="0:0:2"/>
                    </Storyboard>
                </BeginStoryboard>
            </EventTrigger>
        </Ellipse.Triggers>
    </Ellipse>
</Canvas>
```

## Пример решения (фрагменты)

<details>
<summary>✅ Fade In Animation</summary>

```xml
<Window.Resources>
    <Storyboard x:Key="FadeInStoryboard">
        <DoubleAnimation Storyboard.TargetName="titleText"
                         Storyboard.TargetProperty="Opacity"
                         From="0" To="1"
                         Duration="0:0:0.5"
                         BeginTime="0:0:0">
            <DoubleAnimation.EasingFunction>
                <QuadraticEase EasingMode="EaseOut"/>
            </DoubleAnimation.EasingFunction>
        </DoubleAnimation>
        
        <DoubleAnimation Storyboard.TargetName="subtitleText"
                         Storyboard.TargetProperty="Opacity"
                         From="0" To="1"
                         Duration="0:0:0.5"
                         BeginTime="0:0:0.1">
            <DoubleAnimation.EasingFunction>
                <QuadraticEase EasingMode="EaseOut"/>
            </DoubleAnimation.EasingFunction>
        </DoubleAnimation>
        
        <DoubleAnimation Storyboard.TargetName="contentPanel"
                         Storyboard.TargetProperty="Opacity"
                         From="0" To="1"
                         Duration="0:0:0.5"
                         BeginTime="0:0:0.2">
            <DoubleAnimation.EasingFunction>
                <QuadraticEase EasingMode="EaseOut"/>
            </DoubleAnimation.EasingFunction>
        </DoubleAnimation>
        
        <DoubleAnimation Storyboard.TargetName="buttonPanel"
                         Storyboard.TargetProperty="Opacity"
                         From="0" To="1"
                         Duration="0:0:0.5"
                         BeginTime="0:0:0.3">
            <DoubleAnimation.EasingFunction>
                <QuadraticEase EasingMode="EaseOut"/>
            </DoubleAnimation.EasingFunction>
        </DoubleAnimation>
    </Storyboard>
</Window.Resources>

<Window.Triggers>
    <EventTrigger RoutedEvent="Loaded">
        <BeginStoryboard Storyboard="{StaticResource FadeInStoryboard}"/>
    </EventTrigger>
</Window.Triggers>
```
</details>
