# Модуль 1: Практическая работа

## Задание: Создание каркаса WPF-приложения

**Время выполнения:** 3-4 часа  
**Уровень:** Закрепление материала Модуля 1

---

## Описание

Необходимо создать каркас WPF-приложения с использованием всех изученных тем:
- Dependency Properties
- Attached Properties
- Routed Events (Bubbling и Tunneling)

## Требования

### 1. Структура приложения (30 минут)

Создайте проект со следующей структурой:

```
WpfApp/
├── App.xaml / App.xaml.cs
├── MainWindow.xaml / MainWindow.cs
├── Controls/
│   ├── ColorBox.cs (DP: FillColor, BorderColor, CornerRadius)
│   ├── PercentageSlider.cs (DP: Value с валидацией)
│   └── SmartButton.cs (Custom Routed Event)
├── Helpers/
│   └── GlobalKeyInterceptor.cs (Attached Properties + Routed Event)
└── Themes/
    └── Generic.xaml (стили для кастомных контролов)
```

## Пример решения (фрагменты)

<details>
<summary>ColorBox Template</summary>

```xml
<!-- Themes/Generic.xaml -->
<Style TargetType="local:ColorBox">
    <Setter Property="Template">
        <Setter.Value>
            <ControlTemplate TargetType="local:ColorBox">
                <Border x:Name="border"
                        Background="{TemplateBinding FillColor}"
                        BorderBrush="{TemplateBinding BorderColor}"
                        BorderThickness="1"
                        CornerRadius="{TemplateBinding CornerRadius}">
                </Border>
                
                <ControlTemplate.Triggers>
                    <Trigger Property="IsMouseOver" Value="True">
                        <Setter TargetName="border" Property="Background" 
                                Value="{TemplateBinding HoverColor}"/>
                    </Trigger>
                </ControlTemplate.Triggers>
            </ControlTemplate>
        </Setter.Value>
    </Setter>
</Style>
```
</details>

<details>
<summary>PercentageSlider Coercion</summary>

```csharp
private static object CoerceValue(DependencyObject d, object baseValue)
{
    var control = (PercentageSlider)d;
    var value = (int)baseValue;
    int step = control.Step;
    
    // Округление до ближайшего Step
    int rounded = (int)Math.Round((double)value / step) * step;
    return Math.Max(0, Math.Min(100, rounded));
}
```
</details>

<details>
<summary>SmartButton SmartClick</summary>

```csharp
public class SmartClickEventArgs : RoutedEventArgs
{
    public int ClickCount { get; }
    public bool ShiftPressed { get; }
    public bool CtrlPressed { get; }
    public bool AltPressed { get; }

    public SmartClickEventArgs(
        RoutedEvent routedEvent, 
        object source,
        int clickCount,
        bool shift,
        bool ctrl,
        bool alt) : base(routedEvent, source)
    {
        ClickCount = clickCount;
        ShiftPressed = shift;
        CtrlPressed = ctrl;
        AltPressed = alt;
    }
}
```
</details>
