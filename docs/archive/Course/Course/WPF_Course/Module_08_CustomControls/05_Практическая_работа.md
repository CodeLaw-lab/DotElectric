## Требования

### 1. UserControl: PersonCard (45 минут)

Создайте UserControl для отображения персоны:

```xml
<UserControl x:Class="WpfApp.PersonCard">
    <Border BorderBrush="Gray" BorderThickness="1" CornerRadius="5" Padding="10">
        <StackPanel>
            <!-- Аватар -->
            <Image Width="80" Height="80" 
                   Source="{Binding AvatarUrl, RelativeSource={RelativeSource AncestorType=UserControl}}"/>
            
            <!-- Имя -->
            <TextBlock Text="{Binding Name, RelativeSource={RelativeSource AncestorType=UserControl}}"
                       FontWeight="Bold" FontSize="16"/>
            
            <!-- Email -->
            <TextBlock Text="{Binding Email, RelativeSource={RelativeSource AncestorType=UserControl}}"
                       Foreground="Gray"/>
            
            <!-- Кнопка действия -->
            <Button Content="Contact" 
                    Command="{Binding ContactCommand, RelativeSource={RelativeSource AncestorType=UserControl}}"/>
        </StackPanel>
    </Border>
</UserControl>
```

**Dependency Properties:**
- AvatarUrl (ImageSource)
- Name (string)
- Email (string)
- ContactCommand (ICommand)

### 4. Attached Behavior: TextBoxBehaviors (45 минут)

Создайте класс с Attached Behaviors:

```csharp
public static class TextBoxBehaviors
{
```

### 5. Interactivity: EventToCommand (45 минут)

Создайте ViewModel с использованием Interactivity:

```csharp
public class MainViewModel : ObservableObject
{
    [ObservableProperty]
    private string _logText;

    [RelayCommand]
    private void LogMessage(object parameter)
    {
        LogText += $"{parameter}\n";
    }

    [RelayCommand]
    private void MouseMoved(MouseEventArgs e)
    {
        LogText += $"Mouse: {e.GetPosition(null)}\n";
    }
}
```

```xml
<Window xmlns:i="http://schemas.microsoft.com/xaml/behaviors">
    <Grid>
        <!-- Кнопка с командой -->
        <Button Content="Click">
            <i:Interaction.Triggers>
                <i:EventTrigger EventName="Click">
                    <i:InvokeCommandAction Command="{Binding LogMessageCommand}"
                                           CommandParameter="Button clicked!"/>
                </i:EventTrigger>
            </i:Interaction.Triggers>
        </Button>
        
        <!-- Canvas с MouseMove командой -->
        <Canvas>
            <i:Interaction.Triggers>
                <i:EventTrigger EventName="MouseMove">
                    <i:InvokeCommandAction Command="{Binding MouseMovedCommand}"
                                           PassEventArgsToCommand="True"/>
                </i:EventTrigger>
            </i:Interaction.Triggers>
        </Canvas>
        
        <!-- Лог -->
        <TextBox Text="{Binding LogText}" IsReadOnly="True"/>
    </Grid>
</Window>
```

<details>
<summary>✅ LoadingSpinner Generic.xaml</summary>

```xml
<Style TargetType="{x:Type local:LoadingSpinner}">
    <Setter Property="Width" Value="40"/>
    <Setter Property="Height" Value="40"/>
    <Setter Property="Foreground" Value="{DynamicResource AccentBrush}"/>
    
    <Setter Property="Template">
        <Setter.Value>
            <ControlTemplate TargetType="{x:Type local:LoadingSpinner}">
                <Grid x:Name="PART_RotatingElement">
                    <!-- 8 линий -->
                    <Line X1="20" Y1="5" X2="20" Y2="12" Stroke="{TemplateBinding Foreground}" StrokeThickness="3"/>
                    <Line X1="20" Y1="5" X2="20" Y2="12" Stroke="{TemplateBinding Foreground}" StrokeThickness="3" Opacity="0.875">
                        <Line.RenderTransform>
                            <RotateTransform CenterX="20" CenterY="20" Angle="45"/>
                        </Line.RenderTransform>
                    </Line>
                    <!-- Остальные 6 линий с углами 90, 135, 180, 225, 270, 315 -->
                </Grid>
                
                <ControlTemplate.Triggers>
                    <Trigger Property="IsSpinning" Value="True">
                        <Trigger.EnterActions>
                            <BeginStoryboard>
                                <Storyboard>
                                    <DoubleAnimation Storyboard.TargetName="PART_RotatingElement"
                                                     Storyboard.TargetProperty="(UIElement.RenderTransform).(RotateTransform.Angle)"
                                                     From="0" To="360"
                                                     Duration="{Binding SpinDuration, RelativeSource={RelativeSource TemplatedParent}}"
                                                     RepeatBehavior="Forever"/>
                                </Storyboard>
                            </BeginStoryboard>
                        </Trigger.EnterActions>
                    </Trigger>
                </ControlTemplate.Triggers>
            </ControlTemplate>
        </Setter.Value>
    </Setter>
</Style>
```
</details>
