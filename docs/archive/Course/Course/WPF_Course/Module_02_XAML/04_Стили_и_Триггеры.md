# Тема 2.4: Стили и триггеры

### Теория

**Style** — набор настроек свойств для повторного использования.  
**Trigger** — изменение свойств при выполнении условия.

#### Базовая структура Style

```xml
<Style x:Key="MyStyle" TargetType="Button">
    <!-- Setters -->
    <Setter Property="Background" Value="Blue"/>
    <Setter Property="Foreground" Value="White"/>
    
    <!-- Triggers -->
    <Style.Triggers>
        <Trigger Property="IsMouseOver" Value="True">
            <Setter Property="Background" Value="DarkBlue"/>
        </Trigger>
    </Style.Triggers>
</Style>
```

#### Виды триггеров

| Тип | Описание | Пример |
|-----|----------|--------|
| **Trigger** | PropertyTrigger — по свойству | `IsMouseOver=True` |
| **DataTrigger** | По значению binding | `IsActive=True` |
| **MultiTrigger** | Несколько условий | `IsMouseOver=True AND IsEnabled=True` |
| **MultiDataTrigger** | Несколько DataTrigger | `Status=Error AND HasData=True` |
| **EventTrigger** | По событию | `Loaded`, `MouseEnter` |

#### Наследование стилей (BasedOn)

```xml
<Style x:Key="BaseButton" TargetType="Button">
    <Setter Property="Padding" Value="20,10"/>
    <Setter Property="FontSize" Value="14"/>
</Style>

<Style x:Key="PrimaryButton" 
       TargetType="Button" 
       BasedOn="{StaticResource BaseButton}">
    <Setter Property="Background" Value="Blue"/>
</Style>
```

### Примеры кода

#### Пример 1: Простой Style с PropertyTrigger

```xml
<Window.Resources>
    <Style x:Key="StyledButton" TargetType="Button">
        <!-- Базовые свойства -->
        <Setter Property="Background" Value="#0078D4"/>
        <Setter Property="Foreground" Value="White"/>
        <Setter Property="Padding" Value="20,10"/>
        <Setter Property="BorderThickness" Value="0"/>
        <Setter Property="FontSize" Value="14"/>
        <Setter Property="Cursor" Value="Hand"/>
        
        <!-- Анимация при наведении -->
        <Style.Triggers>
            <Trigger Property="IsMouseOver" Value="True">
                <Setter Property="Background" Value="#005A9E"/>
            </Trigger>
            
            <Trigger Property="IsPressed" Value="True">
                <Setter Property="Background" Value="#004080"/>
                <Setter Property="RenderTransform">
                    <Setter.Value>
                        <ScaleTransform ScaleX="0.98" ScaleY="0.98"/>
                    </Setter.Value>
                </Setter>
            </Trigger>
            
            <Trigger Property="IsEnabled" Value="False">
                <Setter Property="Background" Value="#CCCCCC"/>
                <Setter Property="Foreground" Value="#666666"/>
            </Trigger>
        </Style.Triggers>
    </Style>
</Window.Resources>

<StackPanel>
    <Button Content="Нажми меня" Style="{StaticResource StyledButton}"/>
    <Button Content="Отключена" IsEnabled="False" Style="{StaticResource StyledButton}"/>
</StackPanel>
```

#### Пример 2: DataTrigger

```xml
<Window.Resources>
    <Style x:Key="UserListItem" TargetType="ListBoxItem">
        <Setter Property="Padding" Value="10,5"/>
        
        <Style.Triggers>
            <!-- DataTrigger по значению binding -->
            <DataTrigger Binding="{Binding IsAdmin}" Value="True">
                <Setter Property="Background" Value="#FFF3CD"/>
                <Setter Property="FontWeight" Value="Bold"/>
                <Setter Property="ToolTip" Value="Администратор"/>
            </DataTrigger>
            
            <DataTrigger Binding="{Binding IsActive}" Value="False">
                <Setter Property="Foreground" Value="Gray"/>
                <Setter Property="FontStyle" Value="Italic"/>
            </DataTrigger>
        </Style.Triggers>
    </Style>
</Window.Resources>

<ListBox ItemsSource="{Binding Users}"
         ItemContainerStyle="{StaticResource UserListItem}">
    <ListBox.ItemTemplate>
        <DataTemplate>
            <StackPanel>
                <TextBlock Text="{Binding Name}"/>
                <TextBlock Text="{Binding Email}" FontSize="12" Foreground="Gray"/>
            </StackPanel>
        </DataTemplate>
    </ListBox.ItemTemplate>
</ListBox>
```

#### Пример 3: MultiTrigger

```xml
<Style x:Key="SmartButton" TargetType="Button">
    <Setter Property="Background" Value="Blue"/>
    <Setter Property="Foreground" Value="White"/>
    
    <Style.Triggers>
        <!-- MultiTrigger: несколько условий Property -->
        <MultiTrigger>
            <MultiTrigger.Conditions>
                <Condition Property="IsMouseOver" Value="True"/>
                <Condition Property="IsEnabled" Value="True"/>
            </MultiTrigger.Conditions>
            <Setter Property="Background" Value="DarkBlue"/>
            <Setter Property="Cursor" Value="Hand"/>
        </MultiTrigger>
        
        <!-- MultiTrigger с 3 условиями -->
        <MultiTrigger>
            <MultiTrigger.Conditions>
                <Condition Property="IsMouseOver" Value="True"/>
                <Condition Property="IsEnabled" Value="True"/>
                <Condition Property="IsDefault" Value="True"/>
            </MultiTrigger.Conditions>
            <Setter Property="BorderBrush" Value="Yellow"/>
            <Setter Property="BorderThickness" Value="2"/>
        </MultiTrigger>
    </Style.Triggers>
</Style>
```

#### Пример 4: MultiDataTrigger

```xml
<Style x:Key="OrderStatusStyle" TargetType="TextBlock">
    <Style.Triggers>
        <!-- MultiDataTrigger: несколько binding условий -->
        <MultiDataTrigger>
            <MultiDataTrigger.Conditions>
                <Condition Binding="{Binding Status}" Value="Error"/>
                <Condition Binding="{Binding HasRetryOption}" Value="True"/>
            </MultiDataTrigger.Conditions>
            <Setter Property="Background" Value="#F8D7DA"/>
            <Setter Property="Foreground" Value="#721C24"/>
            <Setter Property="Text" Value="Ошибка (можно повторить)"/>
        </MultiDataTrigger>
        
        <MultiDataTrigger>
            <MultiDataTrigger.Conditions>
                <Condition Binding="{Binding Status}" Value="Success"/>
                <Condition Binding="{Binding IsConfirmed}" Value="True"/>
            </MultiDataTrigger.Conditions>
            <Setter Property="Background" Value="#D4EDDA"/>
            <Setter Property="Foreground" Value="#155724"/>
            <Setter Property="FontWeight" Value="Bold"/>
        </MultiDataTrigger>
    </Style.Triggers>
</Style>
```

#### Пример 5: EventTrigger с анимацией

```xml
<Style x:Key="AnimatedButton" TargetType="Button">
    <Setter Property="Background" Value="Blue"/>
    
    <Style.Triggers>
        <!-- EventTrigger на событие Loaded -->
        <EventTrigger RoutedEvent="Loaded">
            <BeginStoryboard>
                <Storyboard>
                    <DoubleAnimation Storyboard.TargetProperty="Opacity"
                                     From="0" To="1"
                                     Duration="0:0:0.5"/>
                </Storyboard>
            </BeginStoryboard>
        </EventTrigger>
        
        <!-- EventTrigger на MouseEnter -->
        <EventTrigger RoutedEvent="MouseEnter">
            <BeginStoryboard>
                <Storyboard>
                    <ColorAnimation Storyboard.TargetProperty="(Button.Background).(SolidColorBrush.Color)"
                                    To="DarkBlue"
                                    Duration="0:0:0.2"/>
                </Storyboard>
            </BeginStoryboard>
        </EventTrigger>
        
        <!-- EventTrigger на MouseLeave -->
        <EventTrigger RoutedEvent="MouseLeave">
            <BeginStoryboard>
                <Storyboard>
                    <ColorAnimation Storyboard.TargetProperty="(Button.Background).(SolidColorBrush.Color)"
                                    To="Blue"
                                    Duration="0:0:0.2"/>
                </Storyboard>
            </BeginStoryboard>
        </EventTrigger>
    </Style.Triggers>
</Style>
```

#### Пример 6: Наследование стилей (BasedOn)

```xml
<Window.Resources>
    <!-- Базовый стиль -->
    <Style x:Key="BaseButton" TargetType="Button">
        <Setter Property="Padding" Value="20,10"/>
        <Setter Property="FontSize" Value="14"/>
        <Setter Property="BorderThickness" Value="0"/>
        <Setter Property="Cursor" Value="Hand"/>
        
        <Style.Triggers>
            <Trigger Property="IsMouseOver" Value="True">
                <Setter Property="Opacity" Value="0.9"/>
            </Trigger>
        </Style.Triggers>
    </Style>
    
    <!-- Primary: наследуется от BaseButton -->
    <Style x:Key="PrimaryButton" 
           TargetType="Button" 
           BasedOn="{StaticResource BaseButton}">
        <Setter Property="Background" Value="#0078D4"/>
        <Setter Property="Foreground" Value="White"/>
    </Style>
    
    <!-- Secondary: наследуется от BaseButton -->
    <Style x:Key="SecondaryButton" 
           TargetType="Button" 
           BasedOn="{StaticResource BaseButton}">
        <Setter Property="Background" Value="#6C757D"/>
        <Setter Property="Foreground" Value="White"/>
    </Style>
    
    <!-- Danger: наследуется от BaseButton -->
    <Style x:Key="DangerButton" 
           TargetType="Button" 
           BasedOn="{StaticResource BaseButton}">
        <Setter Property="Background" Value="#DC3545"/>
        <Setter Property="Foreground" Value="White"/>
    </Style>
    
    <!-- Success: наследуется от PrimaryButton -->
    <Style x:Key="SuccessButton" 
           TargetType="Button" 
           BasedOn="{StaticResource PrimaryButton}">
        <Setter Property="Background" Value="#28A745"/>
    </Style>
</Window.Resources>

<StackPanel>
    <Button Content="Primary" Style="{StaticResource PrimaryButton}"/>
    <Button Content="Secondary" Style="{StaticResource SecondaryButton}"/>
    <Button Content="Danger" Style="{StaticResource DangerButton}"/>
    <Button Content="Success" Style="{StaticResource SuccessButton}"/>
</StackPanel>
```

#### Пример 7: Style без ключа (implicit style)

```xml
<Window.Resources>
    <!-- Style без x:Key применяется ко всем Button в окне -->
    <Style TargetType="Button">
        <Setter Property="Background" Value="#0078D4"/>
        <Setter Property="Foreground" Value="White"/>
        <Setter Property="Padding" Value="15,8"/>
        <Style.Triggers>
            <Trigger Property="IsMouseOver" Value="True">
                <Setter Property="Background" Value="#005A9E"/>
            </Trigger>
        </Style.Triggers>
    </Style>
    
    <!-- Этот стиль тоже применится ко всем Button -->
    <Style TargetType="TextBox">
        <Setter Property="BorderBrush" Value="#0078D4"/>
        <Setter Property="BorderThickness" Value="1"/>
        <Setter Property="Padding" Value="8,4"/>
    </Style>
</Window.Resources>

<StackPanel>
    <!-- Автоматически применяют стили -->
    <Button Content="Кнопка 1"/>
    <Button Content="Кнопка 2"/>
    <TextBox Text="Текст"/>
</StackPanel>
```

### Задачи

#### 🟢 Базовый уровень (45 минут)

**Задача 2.4.1: Простой стиль кнопки**

Создайте стиль `PrimaryButton` для Button:
- Синий фон (#0078D4)
- Белый текст
- Padding 20,10
- Trigger на IsMouseOver (темнее фон)
- Trigger на IsEnabled=False (серый цвет)

**Задача 2.4.2: DataTrigger для TextBox**

Создайте стиль для TextBox:
- Красная рамка при `ValidationError=True`
- Зелёная рамка при `IsValid=True`
- Обычная рамка в остальных случаях

---

#### 🟡 Средний уровень (1.5 часа)

**Задача 2.4.3: Иерархия стилей**

Создайте 4 стиля для Button:
- `BaseButton` (базовый: Padding, FontSize, Cursor)
- `PrimaryButton` (на основе BaseButton + синий цвет)
- `SuccessButton` (на основе PrimaryButton + зелёный)
- `WarningButton` (на основе PrimaryButton + оранжевый)

**Задача 2.4.4: MultiTrigger для карточки**

Создайте стиль для Border (карточка товара):
- Обычное состояние: белая тень
- IsMouseOver: тень больше + курсор рука
- IsSelected: синяя рамка + тень
- MultiTrigger: IsMouseOver AND IsSelected

---

#### 🔴 Продвинутый уровень (3 часа)

**Задача 2.4.5: Система состояний (Visual States)**

Реализуйте стили с состояниями через DataTrigger:
- Состояния: Normal, Loading, Success, Error
- Каждое состояние: свой цвет, иконка, анимация
- Переключение через свойство `State` в ViewModel

**Задача 2.4.6: Adaptive Style**

Создайте адаптивный стиль:
- Меняется от ширины окна (через binding)
- Narrow (<500px): вертикальная раскладка
- Medium (500-1000px): 2 колонки
- Wide (>1000px): горизонтальная раскладка

## Ключевые выводы

✅ **Style** — набор Setter для переиспользования настроек  
✅ **Trigger** — изменение свойств по условию  
✅ **DataTrigger** — реакция на значение binding  
✅ **MultiTrigger** — несколько условий Property  
✅ **MultiDataTrigger** — несколько условий binding  
✅ **EventTrigger** — реакция на события (Loaded, MouseEnter)  
✅ **BasedOn** — наследование стилей  
✅ **Implicit Style** (без x:Key) — применяется ко всем элементам типа
