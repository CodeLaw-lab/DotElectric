# Модуль 2: Практическая работа

## Задание: Создание библиотеки стилей и шаблонов

**Время выполнения:** 4 часа  
**Уровень:** Средний  
**Максимальный балл:** 100

---

## Описание

Необходимо создать полноценную библиотеку стилей и шаблонов для WPF-приложения, включающую:
- Темы (Light/Dark)
- Стили для всех стандартных контролов
- ControlTemplate для кастомизации
- DataTemplate для отображения данных

---

## Требования

### 1. Структура проекта (15 минут)

Создайте структуру файлов:

```
StylesLibrary/
├── App.xaml
├── MainWindow.xaml
├── Resources/
│   ├── Themes/
│   │   ├── LightTheme.xaml
│   │   └── DarkTheme.xaml
│   ├── Styles/
│   │   ├── Buttons.xaml
│   │   ├── TextBoxes.xaml
│   │   ├── ComboBoxes.xaml
│   │   └── DataGrids.xaml
│   └── Templates/
│       ├── CustomButtonTemplate.xaml
│       └── PersonCardTemplate.xaml
```

---

### 2. Темы (30 минут)

**LightTheme.xaml:**
- WindowBackgroundBrush: #FFFFFF
- PanelBackgroundBrush: #F5F5F5
- TextPrimaryBrush: #212121
- TextSecondaryBrush: #757575
- BorderBrush: #E0E0E0
- AccentBrush: #0078D4

**DarkTheme.xaml:**
- WindowBackgroundBrush: #212121
- PanelBackgroundBrush: #303030
- TextPrimaryBrush: #FFFFFF
- TextSecondaryBrush: #BDBDBD
- BorderBrush: #424242
- AccentBrush: #42A5F5

**Требования:**
- Переключение тем через кнопку в Toolbar
- StatusBar отображает текущую тему
- Все ресурсы используют DynamicResource

---

### 3. Стили для Button (45 минут)

Создайте 3 стиля в `Buttons.xaml`:

**PrimaryButton:**
- Фон: AccentBrush
- Текст: Белый
- Padding: 20,10
- FontSize: 14
- Trigger: IsMouseOver (темнее на 10%)
- Trigger: IsPressed (ещё темнее)
- Trigger: IsEnabled=False (серый)

**SecondaryButton:**
- На основе PrimaryButton (BasedOn)
- Фон: #6C757D
- Trigger: IsMouseOver (#5a6268)

**IconButton:**
- На основе PrimaryButton
- Content — иконка + текст
- Padding: 10,8
- Trigger: IsMouseOver (эффект тени)

---

### 4. Стили для TextBox (30 минут)

**NormalTextBox:**
- BorderBrush: BorderBrush
- BorderThickness: 1
- Padding: 8,4
- FontSize: 14
- Trigger: IsFocused (AccentBrush рамка)
- Trigger: ValidationError (красная рамка, толщина 2)

**SearchTextBox:**
- На основе NormalTextBox
- Иконка поиска справа
- Кнопка очистки при вводе

---

### 5. ControlTemplate для Button (45 минут)

Создайте кастомный ControlTemplate:
- Border с CornerRadius=8
- DropShadowEffect (тень)
- ContentPresenter по центру
- Анимация при наведении (ColorAnimation, 0.2s)
- Анимация при нажатии (ScaleTransform)
- Поддержка иконки (через Tag)

---

### 6. DataTemplate для Person (45 минут)

**PersonCardTemplate:**
```csharp
public class Person
{
    public string Name { get; set; }
    public string Email { get; set; }
    public string AvatarUrl { get; set; }
    public bool IsOnline { get; set; }
    public bool IsAdmin { get; set; }
}
```

**Требования:**
- Аватар (Ellipse 60x60)
- Имя (жирный, 16px)
- Email (серый, 13px)
- Индикатор онлайн/офлайн (зелёный/серый)
- Для админа: золотая рамка
- Hover-эффект (тень)

---

### 7. Интеграция в MainWindow (30 минут)

```xml
<Window>
    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="*"/>
            <RowDefinition Height="Auto"/>
        </Grid.RowDefinitions>
        
        <!-- Toolbar -->
        <ToolBar Grid.Row="0">
            <Button Content="🌓 Тема" Command="{Binding ToggleThemeCommand}"/>
            <Separator/>
            <Button Content="Primary" Style="{StaticResource PrimaryButton}"/>
            <Button Content="Secondary" Style="{StaticResource SecondaryButton}"/>
            <Button Content="Icon" Style="{StaticResource IconButton}"/>
        </ToolBar>
        
        <!-- Content -->
        <Grid Grid.Row="1">
            <Grid.ColumnDefinitions>
                <ColumnDefinition/>
                <ColumnDefinition/>
            </Grid.ColumnDefinitions>
            
            <!-- TextBoxes -->
            <StackPanel Grid.Column="0" Margin="20">
                <TextBox Style="{StaticResource NormalTextBox}"
                         Margin="0,0,0,10"/>
                <TextBox Style="{StaticResource SearchTextBox}"/>
            </StackPanel>
            
            <!-- Person List -->
            <ListBox Grid.Column="1" 
                     ItemsSource="{Binding People}"
                     ItemTemplate="{StaticResource PersonCardTemplate}"
                     Margin="20"/>
        </Grid>
        
        <!-- StatusBar -->
        <StatusBar Grid.Row="2">
            <StatusBarItem>
                <TextBlock Text="{Binding CurrentThemeName}"/>
            </StatusBarItem>
        </StatusBar>
    </Grid>
</Window>
```

## Пример решения (фрагменты)

<details>
<summary>LightTheme.xaml</summary>

```xml
<ResourceDictionary xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    
    <Color x:Key="WindowBackgroundColor">#FFFFFF</Color>
    <Color x:Key="PanelBackgroundColor">#F5F5F5</Color>
    <Color x:Key="TextPrimaryColor">#212121</Color>
    <Color x:Key="TextSecondaryColor">#757575</Color>
    <Color x:Key="BorderColor">#E0E0E0</Color>
    <Color x:Key="AccentColor">#0078D4</Color>
    
    <SolidColorBrush x:Key="WindowBackgroundBrush" Color="{StaticResource WindowBackgroundColor}"/>
    <SolidColorBrush x:Key="PanelBackgroundBrush" Color="{StaticResource PanelBackgroundColor}"/>
    <SolidColorBrush x:Key="TextPrimaryBrush" Color="{StaticResource TextPrimaryColor}"/>
    <SolidColorBrush x:Key="TextSecondaryBrush" Color="{StaticResource TextSecondaryColor}"/>
    <SolidColorBrush x:Key="BorderBrush" Color="{StaticResource BorderColor}"/>
    <SolidColorBrush x:Key="AccentBrush" Color="{StaticResource AccentColor}"/>
</ResourceDictionary>
```
</details>

<details>
<summary>PrimaryButton Style</summary>

```xml
<Style x:Key="PrimaryButton" TargetType="Button">
    <Setter Property="Background" Value="{StaticResource AccentBrush}"/>
    <Setter Property="Foreground" Value="White"/>
    <Setter Property="Padding" Value="20,10"/>
    <Setter Property="FontSize" Value="14"/>
    <Setter Property="BorderThickness" Value="0"/>
    <Setter Property="Cursor" Value="Hand"/>
    
    <Style.Triggers>
        <Trigger Property="IsMouseOver" Value="True">
            <Setter Property="Background" Value="#005A9E"/>
        </Trigger>
        <Trigger Property="IsPressed" Value="True">
            <Setter Property="Background" Value="#004080"/>
        </Trigger>
        <Trigger Property="IsEnabled" Value="False">
            <Setter Property="Background" Value="#CCCCCC"/>
            <Setter Property="Foreground" Value="#666666"/>
            <Setter Property="Cursor" Value="Arrow"/>
        </Trigger>
    </Style.Triggers>
</Style>
```
</details>

<details>
<summary>PersonCardTemplate</summary>

```xml
<DataTemplate x:Key="PersonCardTemplate">
    <Border Padding="10" Margin="5" BorderThickness="1" BorderBrush="#E0E0E0">
        <Border.Style>
            <Style TargetType="Border">
                <Style.Triggers>
                    <Trigger Property="IsMouseOver" Value="True">
                        <Setter Property="Effect">
                            <Setter.Value>
                                <DropShadowEffect BlurRadius="15" Opacity="0.3"/>
                            </Setter.Value>
                        </Setter>
                    </Trigger>
                </Style.Triggers>
            </Style>
        </Border.Style>
        
        <Grid>
            <Grid.ColumnDefinitions>
                <ColumnDefinition Width="Auto"/>
                <ColumnDefinition Width="*"/>
            </Grid.ColumnDefinitions>
            
            <Ellipse Grid.Column="0" Width="60" Height="60">
                <Ellipse.Fill>
                    <ImageBrush ImageSource="{Binding AvatarUrl}"/>
                </Ellipse.Fill>
            </Ellipse>
            
            <StackPanel Grid.Column="1" Margin="10,0,0,0">
                <TextBlock Text="{Binding Name}" FontSize="16" FontWeight="Bold"/>
                <TextBlock Text="{Binding Email}" Foreground="Gray" FontSize="13"/>
                
                <StackPanel Orientation="Horizontal" Margin="0,5,0,0">
                    <Ellipse Width="10" Height="10" Fill="#28A745">
                        <Ellipse.Style>
                            <Style TargetType="Ellipse">
                                <Style.Triggers>
                                    <DataTrigger Binding="{Binding IsOnline}" Value="False">
                                        <Setter Property="Fill" Value="#6C757D"/>
                                    </DataTrigger>
                                </Style.Triggers>
                            </Style>
                        </Ellipse.Style>
                    </Ellipse>
                    <TextBlock Margin="5,0,0,0" FontSize="12">
                        <TextBlock.Style>
                            <Style TargetType="TextBlock">
                                <Style.Triggers>
                                    <DataTrigger Binding="{Binding IsOnline}" Value="True">
                                        <Setter Property="Text" Value="Онлайн"/>
                                    </DataTrigger>
                                    <DataTrigger Binding="{Binding IsOnline}" Value="False">
                                        <Setter Property="Text" Value="Офлайн"/>
                                    </DataTrigger>
                                </Style.Triggers>
                            </Style>
                        </TextBlock.Style>
                    </TextBlock>
                </StackPanel>
            </StackPanel>
        </Grid>
    </Border>
</DataTemplate>
```
</details>
