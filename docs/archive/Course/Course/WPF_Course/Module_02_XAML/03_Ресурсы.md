# Тема 2.3: Ресурсы

### Теория

**Ресурсы** — механизм XAML для переиспользования объектов (кисти, стили, шаблоны, данные).

#### Иерархия ресурсов

```
Application.Resources (глобальные)
    ↓
Window.Resources (уровень окна)
    ↓
Layout.Resources (Grid, StackPanel...)
    ↓
Control.Resources (уровень элемента)
```

#### StaticResource vs DynamicResource

| Характеристика | StaticResource | DynamicResource |
|----------------|----------------|-----------------|
| **Когда вычисляется** | При загрузке XAML | В runtime, при запросе |
| **Производительность** | Быстрее | Медленнее |
| **Обновление** | Не обновляется | Автоматически обновляется |
| **Использование** | Статичные ресурсы | Темы, изменяемые ресурсы |

#### ResourceDictionary

```xml
<!-- В отдельном файле -->
<ResourceDictionary xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <!-- Ресурсы -->
</ResourceDictionary>
```

#### MergedDictionaries

```xml
<Application.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <ResourceDictionary Source="Themes/LightTheme.xaml"/>
            <ResourceDictionary Source="Styles/Buttons.xaml"/>
            <ResourceDictionary Source="pack://application:,,,/Assembly;component/Path.xaml"/>
        </ResourceDictionary.MergedDictionaries>
        
        <!-- Локальные ресурсы -->
    </ResourceDictionary>
</Application.Resources>
```

### Примеры кода

#### Пример 1: Уровни ресурсов

```xml
<Window x:Class="WpfApp.MainWindow">
    <!-- Уровень окна -->
    <Window.Resources>
        <SolidColorBrush x:Key="WindowBrush" Color="LightBlue"/>
    </Window.Resources>
    
    <Grid>
        <!-- Уровень Grid -->
        <Grid.Resources>
            <SolidColorBrush x:Key="GridBrush" Color="LightGreen"/>
        </Grid.Resources>
        
        <StackPanel>
            <!-- Уровень элемента -->
            <StackPanel.Resources>
                <SolidColorBrush x:Key="PanelBrush" Color="LightYellow"/>
            </StackPanel.Resources>
            
            <!-- Поиск ресурса: Panel → Grid → Window → Application -->
            <Border Background="{StaticResource PanelBrush}" Height="50"/>
            <Border Background="{StaticResource GridBrush}" Height="50"/>
            <Border Background="{StaticResource WindowBrush}" Height="50"/>
        </StackPanel>
    </Grid>
</Window>
```


```csharp
// ThemeService.cs
public class ThemeService
{
    private readonly ResourceDictionary _lightTheme;
    private readonly ResourceDictionary _darkTheme;
    private bool _isDarkTheme;

    public ThemeService()
    {
        _lightTheme = new ResourceDictionary
        {
            Source = new Uri("Resources/Themes/LightTheme.xaml", UriKind.Relative)
        };

        _darkTheme = new ResourceDictionary
        {
            Source = new Uri("Resources/Themes/DarkTheme.xaml", UriKind.Relative)
        };

        _isDarkTheme = false;
    }

    public void ToggleTheme()
    {
        var app = Application.Current;
        
        if (_isDarkTheme)
        {
            app.Resources.MergedDictionaries.Remove(_darkTheme);
            app.Resources.MergedDictionaries.Add(_lightTheme);
        }
        else
        {
            app.Resources.MergedDictionaries.Remove(_lightTheme);
            app.Resources.MergedDictionaries.Add(_darkTheme);
        }
        
        _isDarkTheme = !_isDarkTheme;
    }
}
```

#### Пример 6: Библиотека ресурсов

```csharp
// ResourcesManager.cs
public static class ResourcesManager
{
    private static readonly Dictionary<string, ResourceDictionary> _cachedDictionaries 
        = new Dictionary<string, ResourceDictionary>();

    public static ResourceDictionary Load(string path)
    {
        if (!_cachedDictionaries.ContainsKey(path))
        {
            var dict = new ResourceDictionary
            {
                Source = new Uri(path, UriKind.RelativeOrAbsolute)
            };
            _cachedDictionaries[path] = dict;
        }
        
        return _cachedDictionaries[path];
    }

    public static void Merge(string path)
    {
        var dict = Load(path);
        if (!Application.Current.Resources.MergedDictionaries.Contains(dict))
        {
            Application.Current.Resources.MergedDictionaries.Add(dict);
        }
    }

    public static void Unmerge(string path)
    {
        if (_cachedDictionaries.TryGetValue(path, out var dict))
        {
            Application.Current.Resources.MergedDictionaries.Remove(dict);
        }
    }
}
```

```csharp
// ThemeService.cs
public interface IThemeService
{
    bool IsDarkTheme { get; }
    void ToggleTheme();
    event Action ThemeChanged;
}

public class ThemeService : IThemeService
{
    private readonly ResourceDictionary _lightTheme;
    private readonly ResourceDictionary _darkTheme;
    private bool _isDarkTheme;

    public bool IsDarkTheme => _isDarkTheme;
    public event Action ThemeChanged;

    public ThemeService()
    {
        _lightTheme = new ResourceDictionary
        {
            Source = new Uri("Resources/Themes/LightTheme.xaml", UriKind.Relative)
        };

        _darkTheme = new ResourceDictionary
        {
            Source = new Uri("Resources/Themes/DarkTheme.xaml", UriKind.Relative)
        };
    }

    public void ToggleTheme()
    {
        var app = Application.Current;
        
        if (_isDarkTheme)
        {
            app.Resources.MergedDictionaries.Remove(_darkTheme);
            app.Resources.MergedDictionaries.Add(_lightTheme);
        }
        else
        {
            app.Resources.MergedDictionaries.Remove(_lightTheme);
            app.Resources.MergedDictionaries.Add(_darkTheme);
        }
        
        _isDarkTheme = !_isDarkTheme;
        ThemeChanged?.Invoke();
    }
}
```

## Ключевые выводы

✅ **Иерархия ресурсов:** Application → Window → Layout → Control  
✅ **StaticResource** — для статичных ресурсов (быстрее)  
✅ **DynamicResource** — для тем и изменяемых ресурсов  
✅ **MergedDictionaries** — подключение внешних словарей  
✅ **Source формат:** относительный путь, pack URI, или URI к сборке  
✅ **Программный доступ:** `FindResource()`, `TryFindResource()`, `Resources[]`
