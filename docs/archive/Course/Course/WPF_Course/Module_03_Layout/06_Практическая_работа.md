# Модуль 3: Практическая работа

## Задание: Создание главного окна приложения

**Время выполнения:** 4 часа  
**Уровень:** Средний  
**Максимальный балл:** 100

---

## Описание

Необходимо создать главное окно приложения с использованием Grid и других layout-контейнеров. Окно должно включать:
- Menu и ToolBar
- Resizable панели (левая, правая)
- Центральную область с TabControl
- StatusBar
- GridSplitter для изменения размеров

---

## Требования

### 1. Структура Grid (30 минут)

Создайте Grid со следующей структурой:

```
┌─────────────────────────────────────────────────────┐
│ Row 0: Menu (Auto)                                  │
├─────────────────────────────────────────────────────┤
│ Row 1: ToolBar (Auto)                               │
├──────────┬──────────────────────┬───────────────────┤
│ Row 2:   │ Col 0    │ Col 1     │ Col 2    │ Col 3 │
│          │ (200px)  │ (Auto)    │ (*)      │(Auto) │
│          │ Left     │ Splitter  │ Center   │Split- │
│          │ Panel    │           │          │ ter   │
│          │          │           │          │       │
│          │          │           │          │Col 4  │
│          │          │           │          │(250px)│
│          │          │           │          │Right  │
│          │          │           │          │Panel  │
├──────────┴──────────────────────┴───────────────────┤
│ Row 3: StatusBar (Auto)                             │
└─────────────────────────────────────────────────────┘
```

**Требования:**
- 5 колонок: 200px, Auto, *, Auto, 250px
- 4 строки: Auto, Auto, *, Auto
- MinWidth/MaxWidth для левой (150-400px) и правой (200-500px) панелей

### 4. Левая панель (30 минут)

Создайте левую панель с TreeView:

```xml
<Border Grid.Column="0" Grid.Row="2"
        Background="{DynamicResource PanelBackgroundBrush}"
        BorderBrush="{DynamicResource BorderBrush}"
        BorderThickness="0,0,1,0">
    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="*"/>
        </Grid.RowDefinitions>
        
        <!-- Заголовок -->
        <TextBlock Grid.Row="0" 
                   Text="Библиотека шаблонов"
                   FontWeight="Bold"

                   Foreground="White"/>
        
        <!-- TreeView -->
        <TreeView Grid.Row="1" Margin="5">
            <TreeViewItem Header="Стандартные шаблоны">
                <TreeViewItem Header="A0 (1189×841)"/>
                <TreeViewItem Header="A1 (841×594)"/>
                <TreeViewItem Header="A2 (594×420)"/>
                <TreeViewItem Header="A3 (420×297)"/>
                <TreeViewItem Header="A4 (297×210)"/>
            </TreeViewItem>
            <TreeViewItem Header="Пользовательские шаблоны">
                <TreeViewItem Header="Шаблон 1"/>
                <TreeViewItem Header="Шаблон 2"/>
            </TreeViewItem>
        </TreeView>
    </Grid>
</Border>
```

### 5. Центральная область (45 минут)

Создайте TabControl с вкладками:

```xml
<TabControl Grid.Column="2" Grid.Row="2" 
            Margin="0"
            Background="White">
    <TabControl.ItemTemplate>
        <DataTemplate>
            <StackPanel Orientation="Horizontal">
                <TextBlock Text="{Binding Title}" Margin="0,0,5,0"/>
                <Button Content="✕" 
                        Padding="2,0"
                        FontSize="10"
                        Command="{Binding CloseTabCommand}"/>
            </StackPanel>
        </DataTemplate>
    </TabControl.ItemTemplate>
    
    <TabControl.ContentTemplate>
        <DataTemplate>
            <Grid Background="White">
                <!-- Canvas для рисования -->
                <Canvas Background="White" ClipToBounds="True">
                    <!-- Пример содержимого -->
                    <Rectangle Canvas.Left="50" Canvas.Top="50"
                               Width="200" Height="150"
                               Fill="LightBlue"
                               Stroke="Blue"
                               StrokeThickness="2"/>
                    <Line X1="100" Y1="100" X2="200" Y2="150"
                          Stroke="Red" StrokeThickness="2"/>
                </Canvas>
            </Grid>
        </DataTemplate>
    </TabControl.ContentTemplate>
</TabControl>
```

**Требования:**
- Кнопка закрытия на каждой вкладке
- Canvas с примерами фигур (Rectangle, Line, Ellipse)
- Белый фон рабочей области

                <RowDefinition Height="*"/>
