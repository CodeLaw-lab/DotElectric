## Подтверждённые решения

| Параметр | Решение |
|----------|---------|
| Создание проекта | **dotnet CLI** |
| Структура папок | **Как в спецификации** (src/DotElectric.TemplateEditor/ + .Tests/) |

## Спринт 3: Tools + ViewModels + Views

### 3.1 Tools (State Pattern)

| Инструмент | Описание |
|------------|----------|
| **ITool** | OnMouseDown(modelPoint), OnMouseMove, OnMouseUp, OnDoubleClick, OnMouseWheel, GetCursor() |
| **SelectTool** | Клик (выбор), Shift+клик (добавление), Ctrl+клик (снятие), перетаскивание (move), Delete, стрелки |
| **DrawingLineTool** | Preview-линия (1 раз создание, обновление свойств), Shift (горизонт/вертик), привязка |
| **DrawingRectangleTool** | Preview-прямоугольник, Shift (квадрат), Ctrl (от центра), привязка |
| **TextTool** | Клик → диалог → создание Text, SingleLine/MultiLine |
| **PanTool** | Среднее колесо мыши, курсор hand (PushTool) |
| **ResizeTool** | Маркеры (2/8/4), Shift (пропорции), Ctrl (от центра), мин 1мм (PushTool) |

**Вращение:** E = +90° по часовой, Shift+E = -90°

**ToolStack:** PushTool/PopTool для временной смены инструмента (Pan, Resize). Базовый инструмент через SetActiveTool().

### 3.2 ViewModels

| ViewModel | Описание |
|-----------|----------|
| **MainViewModel** (Singleton) | OpenedTabs (ObservableCollection), SelectedTab, NewTab, CloseTab, OpenFile, SaveFile |
| **EditorViewModel** (Factory) | Template, SelectedObjects, ActiveTool, Zoom (10%-500%), GridSettings (настраивается, НЕ сериализуется), CommandHistory, IsDirty (через MarkDirty) |
| **EditorViewModelFactory** | Create(), CreateNew(format), CreateFromFile(path) — внедрение через DI |
| **PropertiesViewModel** | Привязка к SelectedObjects, обновление свойств, валидация Key уникальности |
| **TemplateLibraryViewModel** | LoadTemplateInfos() из `%APPDATA%\DotElectric\Templates\`, сортировка по алфавиту. `ObservableCollection<TemplateInfo>` для биндинга. |

### 3.3 Views

| View | Описание |
|------|----------|
| **EditorCanvas** (WPF Canvas) | FromWpfPoint(), ToWpfPoint(), State Pattern (PushTool/PopTool, ActiveTool), Preview reuse (НЕ создавать новый Shape при MouseMove!), маркеры выделения (Rectangle/Ellipse 6×6px, hit-test 12×12px), пунктирная подсветка multi-selection (DashArray="4,2", 1.5px), отрисовка сетки, Zoom + Pan |
| **MainWindow** | TabControl (вкладки с крестиком), ToolBar, Menu, StatusBar (масштаб, формат, статус) |
| **PropertiesPanel** | Динамические поля по типу объекта, конвертеры координат (мм ↔ микроны) |
| **TemplateLibraryView** | ListBox с именами шаблонов (DisplayName, без `.tdel`). Если коллекция пуста — без элементов. |
| **Themes** | LightTheme.xaml / DarkTheme.xaml (DynamicResource), переключение через MergedDictionaries |

## Зависимости между слоями

```
Views ← ViewModels ← Services ← Models
  │         │            │
  └─────────┴────────────┘
        (чистые зависимости, без циклов)
```

| Правило | Описание |
|---------|----------|
| Models НЕ зависят от WPF | Нет `System.Windows.*` импортов (кроме `CultureInfo`) |
| Services (интерфейсы) НЕ зависят от UI | Нет WPF-типов в `I*Service.cs` |
| Commands НЕ зависят от WPF | `ICommand` (Undo/Redo) ≠ `System.Windows.Input.ICommand` |
| Helpers НЕ зависят от UI | Работают только с моделями |

## Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Шрифты ГОСТ не найдены или несвободные | Средняя | Средняя | Fallback на Arial на Этапе 1 |
