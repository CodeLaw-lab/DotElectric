## 6. РЕАЛИЗОВАННЫЙ ИНТЕРФЕЙС (Dark Theme)

### 6.1. Layout

Главное окно построено на базе `Grid` с 5 колонками и 4 строками:

```
┌─────────────────────────────────────────────────────────────────┐
│ Menu (Material Design, тёмная тема)                             │
├─────────────────────────────────────────────────────────────────┤
│ ToolBar (WrapPanel — кнопки переносятся)                        │
├──────────────┬────┬──────────────────────┬────┬────────────────┤
│ Left Panel   │ GS │   TabControl         │ GS │ Right Panel    │
│ ~200px       │    │   (EditorCanvas)     │    │ ~250px         │
│ Min 150px    │    │   (центр, *)         │    │ Min 200px      │
│ Библиотека   │    │                      │    │ Max 500px      │
│ шаблонов     │    │                      │    │                │
│ Max 400px    │    │                      │    │                │
├──────────────┴────┴──────────────────────┴────┴────────────────┤
│ StatusBar (интерактивные сегменты)                              │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2. Компоненты UI

| Компонент | Описание |
|-----------|----------|
| **Menu** | 6 пунктов (Файл, Правка, Вид, Вставка, Инструменты, Справка) с MDI иконками |
| **ToolBar** | WrapPanel — 6 групп кнопок: Файл, Undo/Redo, Копировать/Вставить, Zoom, Печать, Инструменты, Сетка/Привязка, Тема, ComboBox масштаба |
| **Left Panel** | ListBox библиотеки шаблонов (заглушка «Нет загруженных шаблонов») |
| **Center** | TabControl с welcome-экраном (PackIcon + текст) |
| **Right Panel** | ScrollViewer с заглушкой «Выберите объект» (подготовка к PropertyGrid) |
| **StatusBar** | 5 сегментов: Статус (с иконкой), Сетка (toggle), Привязка (toggle), Формат, Масштаб |
| **GridSplitter** | 2 штуки — для изменения ширины левой и правой панелей |
