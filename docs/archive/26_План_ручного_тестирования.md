# ПЛАН РУЧНОГО ТЕСТИРОВАНИЯ (АРХИВНАЯ ВЕРСИЯ)
## DotElectric Template Editor — Этап 1

**Версия:** 1.4 (Sprint 14 — Ориентация листа)
**Дата:** 09 апреля 2026 г.
**Статус:** АРХИВ (НЕ ИСПОЛЬЗОВАТЬ)

## 1. ОБЩИЕ СВЕДЕНИЯ

### 1.1. Цель
Проверить корректность работы всех пользовательских сценариев Этапа 1 перед переходом к Этапу 2 (Редактор УГО).

### 1.2. Область тестирования

| Параметр | Значение |
|----------|----------|
| **Этап** | 1 — Редактор шаблонов листов |
| **Версия** | 1.4 (Sprint 14, 1211 unit-тестов, 0 сбоев) |
| **Платформа** | Windows 10/11, .NET 10, WPF |
| **Время на тестирование** | ~3.5–4 часа |
| **Тестировщиков** | 1–2 человека |

### 1.3. Системные требования
- ОС: Windows 10/11
- .NET 10 Runtime
- Разрешение экрана: от 1280×720

### 1.4. Изменения в Sprint 14

| Компонент | Что изменилось |
|-----------|---------------|
| **Ориентация листа** | Добавлен enum `SheetOrientation` (Portrait/Landscape) |
| **Форматы** | A0–A3 по умолчанию Landscape, A4 по умолчанию Portrait (ГОСТ 2.301) |
| **Меню «Новый шаблон»** | Каждый формат имеет подменю с двумя ориентациями (A4P/A4L, A3P/A3L и т.д.) |
| **Вкладка** | DisplayName включает ориентацию: «A4 (кн.) — Без имени» |
| **StatusBar** | Показывает ориентацию: «A4 (кн.) 210×297 мм» |
| **Сериализация** | `<Orientation>` в XML, обратная совместимость со старыми файлами |
| **Настройки** | `LastUsedSheetOrientation` сохраняется между сессиями |

## 2. СТРУКТУРА ТЕСТ-КЕЙСА

Каждый тест-кейс содержит:

| Поле | Описание |
|------|----------|
| **ID** | Уникальный идентификатор (MT-001, MT-002…) |
| **Название** | Краткое описание |
| **Приоритет** | P0 (критичный), P1 (важный), P2 (дополнительный) |
| **Шаги** | Пошаговая инструкция |
| **Ожидаемый результат** | Что должно произойти |
| **Фактический результат** | Заполняется тестировщиком |
| **Статус** | ✅ PASS / ❌ FAIL / ⏭️ SKIP |
