## 2. РЕАЛИЗОВАННЫЕ КОМПОНЕНТЫ

### 2.1. ViewModels (3 класса)

#### EditorViewModel

**Файл:** `ViewModels/EditorViewModel.cs`

| Свойство/Метод | Описание |
|----------------|----------|
| `TabId` | Уникальный ID вкладки (для автосохранения) |
| `Template` | Модель шаблона (данные) |
| `CommandHistory` | Undo/Redo (50 уровней, MarkDirty callback) |
| `SelectedObjects` | ObservableCollection выделенных объектов |
| `GridSettings` | Настройки сетки (НЕ сериализуются) |
| `IsDirty` | Флаг «грязности» (с последнего сохранения) |
| `FilePath` | Путь к файлу (null для нового шаблона) |
| `DisplayName` | Имя вкладки (с * при IsDirty) |
| `Zoom` / `ZoomPercent` | Масштаб (1.0 = 100%, ограничение 10%-1000%) |
| `ActiveTool` | Текущий активный инструмент (string) |
| `UndoCommand/RedoCommand` | RelayCommand для XAML binding |
| `UndoDisplayName/RedoDisplayName` | "↶ Переместить объект" и т.п. |

**Методы:**
- `MarkDirty()` — пометить как изменённый, обновить DisplayName
- `ClearDirty()` — сбросить флаг (после сохранения), история НЕ очищается
- `SetZoom(double)` — установить зум (с Clamp 0.1-10.0)
- `SetZoomPercent(int)` — установить зум в процентах
- `FitToScreen(canvasWidth, canvasHeight)` — вписать лист в экран
- `SelectSingle/AddToSelection/RemoveFromSelection/ClearSelection/SelectAll` — управление выделением
- `DeleteSelected()` — удалить выделенные объекты (через Undo-команды)
- `MoveSelected(deltaX, deltaY)` — переместить выделенные

**Конструкторы:**
- `EditorViewModel(Template, ITemplateService, GridSettings?)` — новый шаблон
- `EditorViewModel(Template, filePath, ITemplateService, GridSettings?)` — из файла

#### MainViewModel

**Файл:** `ViewModels/MainViewModel.cs`

| Свойство/Метод | Описание |
|----------------|----------|
| `OpenedTabs` | ObservableCollection<EditorViewModel> (вкладки) |
| `SelectedTab` | Активная вкладка |
| `Theme` | Тема оформления (Light/Dark) |

**Команды:**
- `NewTab(format)` — создать пустой шаблон (A0-A4)
- `OpenFileAsync()` — открыть .tdel через OpenFileDialog
- `SaveAsync()` — сохранить активную вкладку
- `SaveAllAsync()` — сохранить все вкладки
- `SaveAsAsync()` — сохранить как
- `CloseTabAsync(tab)` — закрыть вкладку (с диалогом несохранённых изменений)
- `CloseAllTabsAsync()` — закрыть все вкладки
- `ToggleTheme()` — переключить тему

**Бизнес-логика сохранения:**
1. Если нет FilePath → SaveFileDialog
2. Валидация через ITemplateService.Validate
3. При наличии ошибок → диалог подтверждения
4. Бэкап предыдущей версии (если файл существует)
5. Сохранение, обновление FilePath, ClearDirty

### 2.2. Tools (State Pattern — 4 класса + интерфейс)

#### ITool (интерфейс)

**Файл:** `Tools/ITool.cs`

```
OnMouseDown(modelPoint, button, modifiers)  — нажатие мыши
OnMouseMove(modelPoint, button, modifiers)  — перемещение мыши
OnMouseUp(modelPoint, button, modifiers)    — отпускание мыши
OnDoubleClick(modelPoint)                   — двойной клик
OnMouseWheel(delta, modelPoint)            — колесо (зум)
GetCursor() → ToolCursor                   — курсор мыши
```

**ToolCursor enum:** Arrow, Cross, Hand, IBeam, SizeNWSE, SizeNESW, SizeWE, SizeNS

#### SelectTool

**Файл:** `Tools/SelectTool.cs`

| Действие | Поведение |
|----------|----------|
| Клик на объекте | Выделить только этот |
| Shift+клик | Добавить к выделению |
| Ctrl+клик | Снять с выделения |
| Клик в пустое | Снять выделение (без Shift) |
| Перетаскивание | Переместить выделенные (threshold 3мм) |
| Двойной клик по тексту | Inline-редактирование (TODO) |

**Особенности:**
- Hit-test через HitTestHelper (tolerance 5мм для линий)
- Threshold 3мм — чтобы не путать клик с drag
- Курсор: Arrow → Hand (при drag)

#### DrawingLineTool

**Файл:** `Tools/DrawingLineTool.cs`

| Действие | Поведение |
|----------|----------|
| Первый клик | Начальная точка, создание PreviewLine |
| MouseMove | Обновление конца preview-линии |
| Второй клик | Создание линии (AddObjectCommand) |
| Двойной клик | Отменить текущую линию |
| Shift | Ограничение горизонталью/вертикалью |

**Preview-отрисовка:** Line создаётся 1 раз, обновляются только EndMicronsX/Y

#### DrawingRectangleTool

**Файл:** `Tools/DrawingRectangleTool.cs`

| Модификаторы | Поведение |
|-------------|----------|
| Без модификаторов | От угла к углу |
| Shift | Квадрат (равные стороны) |
| Ctrl | От центра |
| Ctrl+Shift | Квадрат от центра |

**Действия:**
- Первый клик → начальный угол + PreviewRectangle
- MouseMove → обновление preview
- Второй клик → создание Rectangle (AddObjectCommand)
- Двойной клик → отмена

#### TextTool

**Файл:** `Tools/TextTool.cs`

| Действие | Поведение |
|----------|----------|
| Клик | Создать Text в точке клика |
| MouseMove | Обновление позиции preview-текста |

**Настройки по умолчанию:**
- Шрифт: ГОСТ Б
- Размер: 3500 микрон (3.5мм)
- Тип: Text
- Содержимое: "Текст"

**Методы:** SetTextType, SetFontSize, SetDefaultContent

## 3. КЛЮЧЕВЫЕ АРХИТЕКТУРНЫЕ РЕШЕНИЯ

### 3.1. EditorViewModel — одна вкладка = один экземпляр

```
MainViewModel
└── OpenedTabs: ObservableCollection<EditorViewModel>
    ├── EditorViewModel[0] — Template, CommandHistory, Zoom, GridSettings
    ├── EditorViewModel[1] — Template, CommandHistory, Zoom, GridSettings
    └── EditorViewModel[2] — ...
```

Каждая вкладка имеет **свою** CommandHistory, **свой** Zoom, **свои** GridSettings.

### 3.2. Checkpoint-подход к сохранению

`ClearDirty()` сбрасывает IsDirty, но **НЕ очищает** CommandHistory. Пользователь может:
1. Сохранить файл
2. Отменить изменения (Undo)
3. Сохранить снова (более раннюю версию)

### 3.3. State Pattern для инструментов

```
EditorCanvas
└── CurrentTool: ITool
    ├── SelectTool → OnMouseDown → HitTest → Select/Drag
    ├── DrawingLineTool → OnMouseDown → PreviewLine → AddObject
    ├── DrawingRectangleTool → OnMouseDown → PreviewRect → AddObject
    └── TextTool → OnMouseDown → Add Text
```

Переключение: `CurrentTool = new SelectTool(editor)`

### 3.4. Preview-отрисовка

Shape создаётся 1 раз (PreviewLine, PreviewRectangle, PreviewText), обновляются **только свойства**. НЕ создаётся новый Shape при каждом MouseMove.

### 3.5. DisplayName с индикатором изменений

```
"A3 — Без имени"     — новый шаблон, нет изменений
"A3 — Без имени *"   — новый шаблон, есть изменения
"my_template.tdel"   — сохранённый файл, нет изменений
"my_template.tdel *" — сохранённый файл, есть изменения
```
