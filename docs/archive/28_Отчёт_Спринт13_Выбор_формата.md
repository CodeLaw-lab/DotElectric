## 3. Архитектурные решения

### 3.1. Подменю вместо диалога

Выбран паттерн подменю «Файл → Новый шаблон ▸ A0/A1/A2/A3/A4/Пользовательский» — пользователь видит все варианты сразу (1 клик), Ctrl+N = последний использованный формат.

### 3.2. Разделение IDialogService и IDialogHostService

`IDialogService` — для MessageBox (ShowError, ShowConfirmation).  
`IDialogHostService` — для модальных окон с кастомными ViewModel.  
Это обеспечивает тестируемость: мокируется отдельно.

### 3.3. Сохранение LastUsedSheetFormat

При каждом вызове `NewTab(format)` формат сохраняется в настройках. Это позволяет Ctrl+N всегда использовать последний выбранный формат.

### 3.4. CreateFromSheet в TemplateService

Новый метод для создания шаблона из готового `Sheet` объекта — нужен для пользовательских форматов.
