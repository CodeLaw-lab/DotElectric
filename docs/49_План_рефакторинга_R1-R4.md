### R1.7 — Исправить time-flaky тесты (DateTime.Now)

**Файлы:**
- `Tests/Models/TemplateTests.cs:158-159` — `Assert.True(metadata.CreatedDate <= DateTime.Now)`
- `Tests/Models/TemplateExtendedTests.cs:80-81` — то же самое
- 14 других файлов с `DateTime.Now` в тестах (создание Metadata)

**Причина:** `Metadata` конструктор использует `DateTime.UtcNow` (строка 38), а тесты проверяют `<= DateTime.Now`. Race через полночь и часовые пояса.

**Вариант A (минимальный):** Заменить assert на:
```csharp
var now = DateTime.UtcNow;
Assert.True(metadata.CreatedDate <= now.AddSeconds(1));
```

**Вариант B (чище, рекомендую):** Внедрить `IDateTimeProvider` в `Metadata`:
```csharp
public Metadata(IDateTimeProvider? dateTimeProvider = null)
{
    var now = (dateTimeProvider ?? new DateTimeProvider()).UtcNow;
    CreatedDate = now;
    ModifiedDate = now;
}
```
В тестах: передавать `Mock<IDateTimeProvider>` или `null` (будет реальное время).

**Также:** Все 16 мест, где тесты создают `new Metadata { CreatedDate = DateTime.Now, ...}`, заменить на `DateTime.UtcNow` (согласованность с конструктором).

```csharp
    private readonly ResizeState _newState;
    private readonly Action? _markDirty;

    public string Name => "Изменить размер";

    public void Execute()
    {
        _object.ApplyResize(_newState);
        _markDirty?.Invoke();
    }

    public void Undo()
    {
        _object.ApplyResize(_oldState);
        _markDirty?.Invoke();
    }
}
```

`ResizeTool.cs` — в `OnMouseUp` использовать `CaptureResizeState()`.

**Проверка:** `dotnet test` — ResizeToolTests, CustomResizeCommand

### R4.6 — Coverage gate (≥75%) в CI

**Файл:** `.github/workflows/ci.yml`

**Что сделать:** После шага `Run Tests` добавить проверку coverage:

```yaml
- name: Check Coverage
  shell: pwsh
  run: |
    $report = Get-ChildItem -Recurse "TestResults/**/coverage.cobertura.xml" | Select-Object -First 1
    if (-not $report) { throw "No coverage report found" }
    [xml]$xml = Get-Content $report.FullName
    $coverage = [math]::Round($xml.CoverageReport.Summary.LineRate * 100, 1)
    $threshold = 75
    if ($coverage -lt $threshold) {
        throw "Coverage $coverage% is below threshold of $threshold%"
    }
    Write-Host "Coverage: $coverage% (threshold: $threshold%) ✅"
```

Также добавить `reportgenerator`:
```yaml
- name: Install reportgenerator
  run: dotnet tool install -g dotnet-reportgenerator-globaltool

- name: Generate coverage HTML
  run: reportgenerator "-reports:TestResults/**/coverage.cobertura.xml" "-targetdir:coveragereport" -reporttypes:Html

- name: Upload Coverage HTML
  uses: actions/upload-artifact@v4
  with:
    name: coverage-html
    path: coveragereport/
```

**Проверка:** CI зелёный
