## Принятые решения по интерфейсу

| # | Область | Решение | Обоснование |
|---|---------|---------|-------------|
| 1 | **Раскладка окна** | **Grid с Row/Column Definitions** | Гибкость, пропорции, поддержка GridSplitter |
| 2 | **Размеры панелей** | **Resizable с GridSplitter** | Пользователь настраивает ширину левой/правой панели |
| 3 | **ToolBar** | **Один ToolBar с WrapPanel** | Кнопки переносятся на следующую строку при нехватке места |
| 4 | **Панель свойств** | **PropertyGrid (классический подход)** | Строки «Координата X: 10.000 мм» с редакторами для каждого типа. Группировка по секциям: «Позиция», «Размеры», «Оформление», «Текст» |
| 5 | **Multi-tab** | **Оба варианта: [+] в табах + кнопка в ToolBar** | Дублирование для удобства. При закрытии последнего таба — рабочая область пустая (welcome-экран НЕ показывается) |
| 6 | **StatusBar** | **Интерактивные сегменты** | Клик по масштабу → popup с вводом %, клик по сетке → toggle, клик по формату → диалог |
| 7 | **Визуальный стиль** | **Material Design** | Плоский дизайн, тени, акцентные цвета (#0078D4) |
| 8 | **Иконки** | **Material Design Icons** | Пакет `MaterialDesignThemes` (NuGet) |
| 9 | **Диалог настроек** | **Модальное окно (Window) 420×440, NoResize** | Открывается через **Файл → Настройки...** → `OpenSettingsCommand` → `WpfDialogHostService` → `SettingsView` (SettingsViewModel). Секции: ОФОРМЛЕНИЕ (тема), СЕТКА (показывать/привязка/шаг), АВТОСОХРАНЕНИЕ (интервал), НОВЫЙ ШАБЛОН (формат/масштаб). Кнопки Сохранить/Отмена. |

## Структура главного окна (Grid)

```
┌─────────────────────────────────────────────────────────────────┐
│ Row 0: Menu (Auto)                                              │
├─────────────────────────────────────────────────────────────────┤
│ Row 1: ToolBar (Auto) — WrapPanel, может занять 2 строки         │
├──────────────┬────────────┬──────────────────┬──────────────────┤
│ Row 2:       │ Col 0      │ Col 1 (GridSplit)│ Col 2            │ Col 3          │
│ Left Panel   │ (Auto)     │                  │ (Star)           │ (Auto)         │
│ ~200px       │ Библиотека │   EditorCanvas   │ Свойства         │ Right Panel    │
│ Min 150px    │ шаблонов   │   (TabControl)   │ ~250px           │                │
│ Max 400px    │            │                  │ Min 200px        │                │
│              │            │                  │ Max 500px        │                │
├──────────────┴────────────┴──────────────────┴──────────────────┤
│ Row 3: StatusBar (Auto) — интерактивные сегменты                │
└─────────────────────────────────────────────────────────────────┘
```

### Grid Row/Column Definitions

```xml
<Grid>
    <Grid.RowDefinitions>
        <RowDefinition Height="Auto"/>     <!-- Menu -->
        <RowDefinition Height="Auto"/>     <!-- ToolBar (WrapPanel) -->
        <RowDefinition Height="*"/>        <!-- Основная область -->
        <RowDefinition Height="Auto"/>     <!-- StatusBar -->
    </Grid.RowDefinitions>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="Auto" MinWidth="150" MaxWidth="400"/>  <!-- Левая панель -->
        <ColumnDefinition Width="Auto"/>     <!-- GridSplitter -->
        <ColumnDefinition Width="*"/>        <!-- Центр (TabControl) -->
        <ColumnDefinition Width="Auto"/>     <!-- GridSplitter -->
        <ColumnDefinition Width="Auto" MinWidth="200" MaxWidth="500"/> <!-- Правая панель -->
    </Grid.ColumnDefinitions>
</Grid>
```

## StatusBar — интерактивные сегменты

```xml
<StatusBar>
    <!-- Масштаб: клик → popup с вводом % -->
    <StatusBarItem>
        <Button Content="{Binding ZoomPercent, StringFormat='Масштаб: {0}%'}"
                Command="{Binding ShowZoomDialogCommand}"/>
    </StatusBarItem>
    <Separator/>
    
    <!-- Формат листа: клик → диалог изменения -->
    <StatusBarItem>
        <Button Content="{Binding SheetFormat, StringFormat='{0} ({1}×{2})'}"
                Command="{Binding ShowSheetDialogCommand}"/>
    </StatusBarItem>
    <Separator/>
    
    <!-- Сетка: клик → toggle -->
    <StatusBarItem>
        <ToggleButton Content="Сетка"
                      IsChecked="{Binding GridVisible}"
                      Command="{Binding ToggleGridCommand}"/>
    </StatusBarItem>
    <Separator/>
    
    <!-- Привязка: клик → toggle -->
    <StatusBarItem>
        <ToggleButton Content="Привязка"
                      IsChecked="{Binding SnapEnabled}"
                      Command="{Binding ToggleSnapCommand}"/>
    </StatusBarItem>
    <Separator/>
    
    <!-- Статус (текст, не кликабельный) -->
    <StatusBarItem>
        <TextBlock Text="{Binding StatusMessage}"/>
    </StatusBarItem>
</StatusBar>
```

## Multi-tab — вкладки

```xml
<TabControl ItemsSource="{Binding OpenedTabs}" SelectedItem="{Binding SelectedTab}">
    <TabControl.ItemTemplate>
        <DataTemplate>
            <StackPanel Orientation="Horizontal">
                <TextBlock Text="{Binding Title}" Margin="0,0,8,0"/>
                <Button Content="✕" Command="{Binding CloseTabCommand}" 
                        Padding="2,0,2,0" FontSize="10"/>
            </StackPanel>
        </DataTemplate>
    </TabControl.ItemTemplate>
    <TabControl.ContentTemplate>
        <DataTemplate>
            <local:EditorCanvas DataContext="{Binding}"/>
        </DataTemplate>
    </TabControl.ContentTemplate>
    <!-- Кнопка [+] в конце -->
    <TabControl.ItemsPanel>
        <ItemsPanelTemplate>
            <StackPanel Orientation="Horizontal"/>
        </ItemsPanelTemplate>
    </TabControl.ItemsPanel>
</TabControl>
```

## MaterialDesignThemes — интеграция

### В App.xaml

```xml
<Application.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <!-- Material Design -->
            <materialDesign:CustomColorTheme BaseTheme="Light" 
                                             PrimaryColor="#0078D4" 
                                             SecondaryColor="#66BB6A"/>
            <ResourceDictionary Source="pack://application:,,,/MaterialDesignThemes.Wpf;component/Themes/MaterialDesignTheme.Defaults.xaml"/>
            
            <!-- Темы приложения -->
            <ResourceDictionary Source="Resources/Styles/LightTheme.xaml"/>
            <ResourceDictionary Source="Resources/Styles/DarkTheme.xaml"/>
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</Application.Resources>
```

### В .csproj

```xml
<ItemGroup>
    <PackageReference Include="MaterialDesignThemes" Version="5.*" />
    <PackageReference Include="MaterialDesignColors" Version="3.*" />
</ItemGroup>
```

## Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| MaterialDesignThemes конфликтует с кастомными темами | Средняя | Средняя | Использовать MDI только для иконок, стили — кастомные |
| WrapPanel в ToolBar не обновляет layout | Низкая | Низкая | Test при создании, fallback на два ToolBar'а |
| GridSplitter ломает layout при минимальном окне | Средняя | Средняя | MinWidth/MaxWidth на колонках, тестирование |
| PropertyGrid слишком сложный для WPF | Средняя | Средняя | Простая реализация через ItemsControl + DataTemplate |
